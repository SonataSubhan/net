BÖLMƏ 1

R1 (Router) — IP, SSH və ACL Konfiqurasiyası

R1 üzərinə klikləyin → CLI bölməsinə keçin → Enter vurun → aşağıdakı bloku köçürüb daxil edin. 


enable
configure terminal
hostname R1cihazın adı — topologiyada R1-dən fərqli yazılıbsa (məs. R0, Router1) ona uyğun dəyiş
interface Gig0/0port adı cihazdan asılıdır — bəzi modellərdə Gig0/0/0 ola bilər, topologiyaya bax
description Connection to Switchdescription mətni tapşırıqda fərqli istənilə bilər — tam olaraq orda yazılan kimi yaz
ip address 192.168.10.254 255.255.255.0IP və mask — öz şəbəkənin son istifadə oluna bilən ünvanı və mask-ı
no shutdown
exit
ip domain name exam.comdomain adı tapşırıqda fərqli verilə bilər (məs. lab.com, cisco.com)
crypto key generate rsa
1024RSA bit ölçüsü — tapşırıqda 512/2048 kimi fərqli ədəd istənilə bilər, orda yazılanı yaz
username admin password administifadəçi adı/parol — tapşırıqda fərqli verilibsə (məs. cisco/cisco), o cür yaz
line vty 0 4
transport input ssh
exit
ip access-list extended R1_SSH_ACLACL adı sərbəstdir, istənilən ad qoya bilərsən, sadəcə aşağıda eyni adı istifadə et
permit tcp host 192.168.10.1 host 192.168.10.254 eq 22yalnız PC1-in IP-si (SSH-ə icazəli PC) və R1-in öz IP-si
permit icmp 192.168.10.0 0.0.0.255 host 192.168.10.254şəbəkə ünvanı + wildcard mask (mask-ın tərsi) — şəbəkən fərqlidirsə bunu da dəyiş
exit
interface Gig0/0
ip access-group R1_SSH_ACL in
exit
exit
write
BÖLMƏ 2

Sw1 (Switch) — PortFast, BPDU Guard, DHCP Snooping, SSH və ACL

Sw1 üzərinə klikləyin → CLI bölməsinə keçin → Enter vurun → aşağıdakı bloku köçürüb daxil edin. 


enable
configure terminal
hostname Sw1cihazın adı topologiyada fərqli yazılıbsa (məs. Switch0, SW1) ona uyğun dəyiş, böyük/kiçik hərflərə diqqət et
interface fa0/1
description PC1hansı portda hansı PC bağlıdırsa, description-ı ona uyğun yaz; tapşırıqda hər port üçün ayrıca description tələb olunursa, hər biri üçün eyni qaydada təkrarla (fa0/2 → PC2, fa0/3 → PC3, fa0/4 → PC4)
exit
interface vlan 1
ip address 192.168.10.5 255.255.255.0Sw1-in VLAN1 IP-si — öz cədvəlinə uyğun (şəbəkənin 5-ci usable ünvanı)
no shutdown
exit
ip default-gateway 192.168.10.254R1-in interfeys IP-si ilə eyni olmalıdır
interface range fa0/1-4PC-lərin bağlı olduğu port aralığı — topologiyada fərqli port nömrələri varsa (məs. fa0/1-2 və ya fa0/5-8) ona uyğun dəyiş
spanning-tree portfast
spanning-tree bpduguard enable
exit
ip dhcp snooping
ip dhcp snooping vlan 1VLAN nömrəsi — tapşırıqda fərqli VLAN istənilibsə (məs. VLAN 10) dəyiş
interface Gig0/0/1DHCP server/router-ə bağlı etibarlı port — cihaz modelinə görə Gig0/1 və ya Gig0/0/1 ola bilər, topologiyaya bax
ip dhcp snooping trust
exit
ip domain name exam.comR1 ilə eyni domain adı olmalıdır, əgər tapşırıqda fərqli yazılıbsa onu istifadə et
crypto key generate rsa
1024RSA bit ölçüsü — R1-dəki ilə eyni olmalıdır, tapşırıqda dəyişibsə oranı dəyiş
username admin password administifadəçi adı/parol — tapşırıqda fərqlidirsə dəyiş
line vty 0 15switch-lərdə adətən 0-15 (16 sessiya), router-də 0-4 idi — bu normaldır, dəyişmə
transport input ssh
exit
ip access-list extended SW1_SSH_ACLACL adı sərbəstdir
permit tcp host 192.168.10.2 host 192.168.10.5 eq 22yalnız PC2-nin IP-si (SSH-ə icazəli PC) və Sw1-in VLAN1 IP-si
permit icmp 192.168.10.0 0.0.0.255 host 192.168.10.5şəbəkə ünvanı + wildcard mask — şəbəkən fərqlidirsə dəyiş
exit
interface vlan 1
ip access-group SW1_SSH_ACL in
exit
exit
write

Ümumi qeyd: Sarı rənglə işarələnmiş (belə) hər bir dəyər — IP ünvanları, hostname-lər, description mətnləri, port nömrələri, ACL adları, domain adı, RSA ölçüsü, istifadəçi adı/parol, VLAN nömrəsi — sənin öz tapşırığındaki topologiya və cədvələ görə dəyişə bilər.
